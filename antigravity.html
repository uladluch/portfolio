Connect Gemini CLI

Bridge your mobile app with Gemini CLI on your computer. One project folder at a time on this machine — stop the bridge (Ctrl+C) before starting it in another folder.

1. Open Terminal
Install Gemini CLI (if needed)
npm i -g @google/gemini-cli
Sign in to your Google account (if needed)
gemini
2. Go to your project folder
Gemini runs in this directory. Example:
cd /path/to/your-project
3. Start the bridge (only one terminal)
npx antigravity-gemini-bridge@latest
If you see “port in use”, stop the other bridge with Ctrl+C first. To switch folders: disconnect the project in the iOS app (Sessions), then repeat from step 2.
4. Open this page on the same Mac (Safari/Chrome)
With the bridge already running in Terminal, this tab connects to it automatically. If you see a connection error, start the bridge first, then use “Try again” below.
