Connect Claude Code

Bridge your mobile app with Claude Code CLI on your computer.

1Open Terminal
 Install Claude Code (if needed)
curl -fsSL https://claude.ai/install.sh | sh
 Log in (if needed)
claude
2Start the bridge
npx claude-code-mobile-bridge@latest
3Tap Connect below
